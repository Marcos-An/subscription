## Run it locally

Clone the project

```bash
  git clone git@github.com:Marcos-An/subscription.git
```

Install dependencies

```bash
   yarn || npm install
```

Start the server

```bash
  yarn dev
```
 

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
   

## Run Cypress test

```bash
   yarn run cypress:open || npm run cypress:open
```
